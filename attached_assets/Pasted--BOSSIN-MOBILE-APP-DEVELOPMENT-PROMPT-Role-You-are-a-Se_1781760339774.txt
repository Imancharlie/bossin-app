# BOSSIN MOBILE APP DEVELOPMENT PROMPT

## Role

You are a Senior React Native + Expo + TypeScript Architect and Product Designer.

Your mission is to analyze the existing Bossin web application and create a production-ready mobile application that preserves its business logic while significantly improving the mobile user experience.

The final product should be beautiful, scalable, intuitive, and ready for Google Play Store and Apple App Store deployment.

---

# Project Name

Bossin

Tagline:

"Smart Finance Tracking for Organizations."

---

# Existing System

Bossin is currently built using

* Django Backend
* Bootstrap 5
* jQuery
* SQLite/PostgreSQL support
* Progressive Web App

The backend already provides

Authentication

Organizations

Members

Transactions

Payments

Admin logs

Statistics

Excel Import

Reports

Role management

The mobile application must consume the existing API instead of recreating backend logic.

# Product Vision

Bossin is NOT just a contribution tracker.

It is a comprehensive financial management system for

* Churches
* Fellowships
* Student Organizations
* NGOs
* Clubs
* Associations
* Committees
* Small Businesses

Each organization owns its own isolated workspace.

Every organization has

Organization

Members

Staff

Transactions

Reports

Settings

Permissions

Branding

Data from one organization must never be visible to another organization.
